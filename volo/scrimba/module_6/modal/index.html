Main idea of Modal is to have:

  • An overlay div that is positioned as absolute but at the same time is displayed as none
  • An overlay has to have top, right, bottom, left to be 0 and backgroundColor to be set with alpha canal(opacity)
  • Inside of that Overlay div, you have to create div with modal that is relative, positioned in center
  • 2 btns should be created - open & close modal that will updating display css property of overlay
