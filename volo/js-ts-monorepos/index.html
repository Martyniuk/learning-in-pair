Main Concepts

  • Setting up project with "packages" folder inside
  • Adding workspaces to root package.json file
  • Setting up base linter config & per package
  • Setting up base ts-config & ts-config per package
  • Adding tsconfig.settings on packages (src/packages/!here!) level
  • Adding lerna and config for it
  • Setting up scripty, creating scripts folder on top level with sctipts.sh. Don't forget about `chmod -R +x scripts`.
  • Adding commitlit with proper dependencies and config file
  • Covering internal dependencies with help of lerna --scope command
  • Generating api-report with help of @microsoft/api-extractor & generating docs with @microsoft/api-documenter
